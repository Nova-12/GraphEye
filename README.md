# GraphEye

This project is made of two subprojects: `core` and `web`.

## GraphEye-core

### Requirements

1. Spark (`SPARK_HOME` environment varaible needed)
2. Maven
3. JDK (1.7 or later)

### How to use

```
make
./run.sh <arguments>
```

### Spark monitor

`http://localhost:4040`
